# Smart Price Tracker

一個使用 Flask 建立的商品價格監控系統，可以新增商品網址、即時查詢價格，並記錄歷史價格。

## 功能

- 新增商品網址
- 刪除商品
- 即時價格查詢
- 歷史價格紀錄
- Flask Web Dashboard

## 技術

- Python
- Flask
- BeautifulSoup
- Requests
- SQLite
- HTML / CSS / JavaScript
